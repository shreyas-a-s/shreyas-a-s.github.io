Oneclick CMS Deployer

Installer that automates the deployment of Drupal (a CMS) on Linux servers. Utilizes bash script as the backbone with some PHP to ease the creation of CMS modules. Incorporates many error checking functionality so that even non-techie users could deploy a CMS under minutes.

Linux Tweaks

Installer that can be used to customise Linux distribution that you are using. Works on any distro of Linux that is based on either Debian, Ubuntu, Arch Linux, RedHat Enterprise Linux or NixOS. All the customisations are optional and can be deployed based on user choice.

Linux QEMU

Bash script that can be used to setup virtualization on Linux operating system. Currently works on Linux distributions based on Debian, Ubuntu or Arch Linux.

Kodi on Raspberry Pi

Walkthrough guide to install and setup autostart of Kodi on RaspberryPi B+. Along with that, this includes the how-to on installing third-party wifi drivers for wifi-dongle in Linux, setting up an access point on raspberry pi so that it can be controlled directly from your smartphone as well as some raspberry pi tweaks to make the experience a bit better.

Pipepipe to Newpipe

Scripts to convert Database from PipePipe Android app to be compatible with NewPipe app. Includes two scripts, one written using bash and the other using Python.

Dotfiles

A collection of Linux configuration files that can be used to customise Linux to become way better. This project was initially started as a way for me to make re-installation easier. But over the course of time it turned to be a place from which anybody can take inspiration as well as code to customise their Linux experience.
